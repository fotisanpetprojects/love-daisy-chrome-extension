# Love Daisy Privacy Policy

Love Daisy does not collect, store, sell, or share personal data.

The extension runs locally in the browser. It activates only when the user clicks the extension icon, then injects a temporary daisy game overlay into the current tab.

Love Daisy does not transmit browsing activity, page content, user input, or any other data to external servers.

The extension uses Chrome's `activeTab` and `scripting` permissions only to display the local game overlay after the user clicks the extension icon.
